# Portafolio Personal - César 

Este es el repositorio de mi portafolio web personal, desarrollado como proyecto final para la materia de Diseño Web. Es una **Single-Page Application (SPA)** construida con **React** y **Vite**, diseñada para mostrar mis proyectos de una manera moderna y minimalista.

---

## Características Principales

Este portafolio está construido como una sola página con scroll suave y un diseño totalmente responsivo.

* **Diseño Minimalista:** Una interfaz limpia y profesional que usa una paleta de colores personalizada (Negro, Blanco, Fucsia y Verde).
* **100% Responsivo:** Adaptable a dispositivos móviles, incluyendo un menú de navegación tipo "hamburguesa".
* **Sección de Proyectos:** Una galería en formato de tarjetas donde muestro los trabajos que he realizado, con enlaces directos a la demo (GitHub Pages) y al código fuente (mediante repositorios en GitHub).
* **Componentes Reutilizables:** Desarrollado bajo la filosofía de componentes de React.

---

## Tecnologías Utilizadas

Este proyecto fue construido desde cero utilizando las siguientes tecnologías:

* **React (v18+)**: Para la construcción de la interfaz de usuario.
* **React Hooks**: `useState` para el menú móvil y `useEffect`.
* **Vite**: Como herramienta de empaquetado y servidor de desarrollo.
* **CSS Moderno**:
    * Flexbox y CSS Grid para un layout robusto.
    * Media Queries para el diseño responsivo.
* **GitHub Pages**: Para el despliegue y hosting gratuito.
